Core Components

The drone is built around the ESP32 NodeMCU32S microcontroller, providing both processing power and Wi-Fi connectivity. It features four DC motors with fan-style propellers, controlled through transistors without traditional ESCs.

Flutter Control App

Mobile Controller

The custom Flutter app provides intuitive control for the drone with dual on-screen joysticks. It communicates wirelessly with the ESP32 over Wi-Fi, sending commands to the IP address 192.168.1.7.

Features

  • Dual-joystick control (throttle/yaw and pitch/roll)
  • Wi-Fi connectivity with configurable IP
  • Responsive UI with smooth animations
  • Future plans for Tony Stark-style GUI
