G1 AI Assistant

Voice-controlled personal assistant inspired by Jarvis

A Raspberry Pi-powered AI assistant that controls smart home devices, responds to voice commands, and learns from user behavior.

Key Features

Voice Control

Responds to natural voice commands using advanced speech recognition powered by Whisper.cpp.

# Example command
"G1, turn on the lights"

Smart Home Control

Controls LED strips and relays through voice commands or automated triggers.

# Light control scripts
os.system('python turn_on.py')

Self-Reactive

Automatically turns off lights after inactivity and can trigger time-based events.

# Inactivity detection
if inactive_time > 30:
  turn_off_lights()

Voice Feedback

Provides audible responses using espeak text-to-speech on Raspberry Pi.

# Voice response
os.system('espeak "Lights turned on"')

Audio Transcription

Uses sounddevice library to capture and transcribe audio in real-time.

# Audio capture
with sd.InputStream(
  callback=audio_callback,
  samplerate=16000
)

Shutdown Triggers

Can safely shutdown the system or restart services through voice commands.

# Shutdown command
if "shutdown" in command:
  os.system('sudo shutdown now')

Technology Stack

Core Components

Hardware

  • Raspberry Pi 4 (main controller)
  • Relay module for device control
  • LED strips with power supply
  • USB microphone for voice input

Software

  • Python (main programming language)
  • Whisper.cpp for speech-to-text
  • espeak for text-to-speech
  • sounddevice for audio capture

System Architecture

# Main control flow
1. Capture audio with sounddevice
2. Transcribe with Whisper.cpp
3. Process command (lights, shutdown, etc.)
4. Respond with espeak
5. Monitor for inactivity
